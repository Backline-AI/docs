## Overview

The Slack integration enables Backline to send real-time notifications about vulnerabilities, remediation activities, and security alerts directly to your team's Slack workspace. Stay informed without switching between applications.

## What You Can Do

With the Slack integration, you can:
- Receive notifications about new critical vulnerabilities
- Get updates on remediation progress
- Monitor SLA violations in real-time
- Be alerted when pull requests are created
- Track security milestones and achievements

## Prerequisites

Before connecting Slack, ensure you have:
- A Slack workspace
- Permission to install apps in your workspace
- Access to the channels where you want notifications

## Connecting Slack

<Steps>
  <Step title="Navigate to Integrations">
    Go to the Integration Hub from the main menu.
  </Step>
  <Step title="Select Slack">
    Find and click on the Slack integration card.
  </Step>
  <Step title="Authorize">
    Click **Connect** and log in to your Slack workspace.
  </Step>
  <Step title="Grant Permissions">
    Review and approve the requested permissions for Backline.
  </Step>
  <Step title="Choose Channel">
    Select the channel(s) where you want to receive notifications.
  </Step>
  <Step title="Complete">
    Finish the setup and return to Backline.
  </Step>
</Steps>

## Notification Types

You can configure Slack to receive:
- New vulnerability alerts
- Critical security findings
- Remediation status updates
- SLA violation warnings
- Pull request creation notifications
- Remediation completion confirmations

## Managing Notifications

### Customizing Alerts
Configure which notifications to receive:
1. Open the Integration Hub
2. Click on the Slack integration
3. Navigate to notification settings
4. Toggle specific notification types on or off

### Changing Channels
To send notifications to different channels:
1. Update your Slack integration settings
2. Select new target channels
3. Save your changes

### Disconnecting
To remove the Slack integration:
1. Go to the Integration Hub
2. Click on the Slack integration
3. Select **Disconnect**
4. Confirm the disconnection

<Note>
  You can reconnect Slack at any time without losing your notification preferences.
</Note>
